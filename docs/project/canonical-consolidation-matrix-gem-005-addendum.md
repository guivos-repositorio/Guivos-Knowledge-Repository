## 4. Separações canônicas adicionadas

- progresso não é reconhecimento;
- reconhecimento não é recompensa econômica;
- ponto não é dinheiro;
- ponto não é valor humano;
- emissão não é pagamento;
- resgate não é saque;
- recompensa não é prova de impacto;
- patrocinador não é autoridade sobre evidência;
- cobertura candidata não é cobertura confirmada;
- expiração econômica não apaga histórico;
- sinal de fraude não é fraude comprovada;
- horas de voluntariado não são impacto comprovado.

## 5. Limites preservados

Nenhuma quantidade, valor monetário, taxa de conversão, orçamento, cashback, saque, transferência, negociação, token, criptomoeda, blockchain, patrocinador, parceiro, contrato, regulamento, ledger, carteira, teste, código ou produção é autorizado.

## 6. Próxima dependência

O GEM-006 poderá utilizar as famílias, fontes, responsabilidades e limites do GEM-005 para definir a economia de parceiros somente após aprovação específica.
