## 4. Separações canônicas adicionadas

- GMV não é receita;
- caixa não é reconhecimento;
- receita não é margem ou lucro;
- repasse não é receita integral;
- capital não é receita operacional;
- pontos não são receita;
- pagador, beneficiário e recebedor são papéis distintos;
- família admissível não é mecanismo aprovado;
- mecanismo aprovado para teste não é lançamento;
- pagamento não concede autoridade.

## 5. Limites preservados

Nenhum preço, percentual, comissão específica, plano, contrato, parceiro, sistema de pagamento, tratamento fiscal, reconhecimento contábil, código ou produção é autorizado.

## 6. Próxima dependência

O GEM-004 poderá utilizar as famílias conceitualmente admissíveis para definir gratuito e pago, somente após aprovação específica.
