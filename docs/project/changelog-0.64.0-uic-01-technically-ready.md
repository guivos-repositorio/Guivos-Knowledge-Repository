# Changelog 0.64.0 — UIC-01 Technically Ready for Implementation

## Added

- arquitetura de armazenamento multimodal;
- política de retenção, exclusão, backup e restore;
- política de busca e indexação sensível;
- regras para embeddings e derivados;
- matriz de 30 guardrails técnicos;
- matriz de atores, autoridades e estados;
- requisitos não funcionais;
- SLOs candidatos da Onda 0;
- threat model com 26 ameaças;
- estratégia final de testes;
- matriz de 18 evidências mínimas;
- gates de domínio, dados, segurança, qualidade e governança;
- registro de riscos residuais;
- ADR-CC-001 a ADR-CC-005;
- Unidade de Engenharia 0.5.0;
- Engineering Handoff 0.5.0;
- Product Architecture 1.36.0;
- Roadmap 11.17.0;

- marco vigente de M5.15 para M5.16;
- ponto de retomada para planejamento da Onda 0.

## Resolved

- `UIC01-GAP-003 — armazenamento multimodal`;
- `UIC01-GAP-008 — busca e indexação sensível`;
- `UIC01-GAP-010 — matriz técnica dos guardrails`.

Os três gaps são encerrados por definição técnica. A implementação deverá produzir evidências reais.

## Not authorized

- lançamento;
- contratação automática de tecnologia;
- topologia definitiva;
- tratamento irrestrito de dados;
- uso de índice ou derivado como sistema de registro;
- confirmação ou autorização pela Intelligence.
