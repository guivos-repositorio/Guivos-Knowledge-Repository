# Changelog 0.65.0 — Planejamento da Onda 0 da UIC-01

- plano executável da Onda 0;
- oito incrementos W0-01 a W0-08;
- backlog de 80 histórias verificáveis;
- plano das 16 dependências mínimas;
- ambientes Local, Test, Integration, Security Lab, Performance Lab e Internal Trial;

- seis POCs;
- plano das 18 evidências;
- critérios de validade, armazenamento e gate das evidências;
- registro inicial de 20 riscos;
- níveis Stop-L1 a Stop-L4;
- protocolo de exceções e retomada;
- dez dossiers tecnológicos ADR-TCC-001 a ADR-TCC-010;

## Changed

- estado da UIC-01 de `Technically ready for implementation` para `Wave 0 implementation planned`;
- estado do Handoff de 0.5.0 para 0.6.0;
- marco vigente de M5.16 para M5.17;
- ponto de retomada para proposta de execução controlada do W0-01.

## Clarified

- readiness arquitetural permanece em 100%;
- execução permanece em 0%;
- planning complete não autoriza execution;
- dossier tecnológico não é ADR aceito;
- ambiente definido não é ambiente provisionado;
- POC especificada não é POC validada;
- evidência planejada não é evidência produzida;
- owner lógico não é owner nominal.

- VAL-002 2.1.0;

## Not authorized

- execução do W0-01;
- criação de código ou POC;
- uso de dados pessoais reais;
- Internal Trial;
- produção;
- lançamento;
- contratação automática;
- escolha de stack sem decisão formal;
- avanço automático das demais UICs.
