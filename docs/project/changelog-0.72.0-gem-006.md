## Decisões

- parceiro, papel, relacionamento, oferta e permissão são distintos;
- aprovação não significa ativação;
- pagamento não substitui relevância ou qualidade;
- receita, repasse, recurso vinculado e cobertura são separados;
- dados dependem de finalidade;
- saída preserva obrigações;
- parceiro estratégico não possui autoridade automática.
