# Métricas do Gratuito, Subsídios e Acesso Financiado

## 1. Objetivo

Observar a preservação do baseline gratuito e a legitimidade de subsídios sem reduzir pessoas a custo, confundir gratuidade com ausência de recurso ou transferir autoridade ao financiador.

## 2. Indicadores candidatos

| Indicador | Fórmula conceitual |
|---|---|
| cobertura do baseline gratuito | componentes essenciais financiados / componentes essenciais avaliados |
| continuidade do gratuito | períodos com baseline íntegro / períodos observados |
| custo do gratuito por unidade | custos elegíveis do baseline / unidades elegíveis servidas |
| acesso financiado efetivo | beneficiários que receberam valor / acessos financiados elegíveis |
| suficiência de subsídio | recursos elegíveis disponíveis / obrigações elegíveis do subsídio |
| dependência de subsídio | custos cobertos por subsídio / custos elegíveis do escopo |
| concentração do financiador | exposição ao maior financiador / financiamento elegível |
| taxa de encerramento seguro | programas encerrados com obrigações tratadas / programas encerrados |

## 3. Segmentações

Fonte, finalidade, produto, programa, população elegível, duração, restrição, recorrência e condição de saída.

## 4. Guardrails

- beneficiário não será exposto como contrapartida publicitária;
- patrocinador não definirá elegibilidade individual ou relevância funcional;
- acesso financiado não reduzirá direitos ou privacidade;
- subsídio sem owner, revisão, cobertura ou saída será classificado como frágil;
- degradação silenciosa do gratuito será bloqueada.
