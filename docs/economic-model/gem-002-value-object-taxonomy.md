# Taxonomia de Objetos de Valor

## 1. Objetivo

Definir os elementos que podem circular entre atores econômicos sem reduzir valor a dinheiro, produto ou dado.

## 2. Definição

Objeto de valor é o benefício, recurso, direito, capacidade, experiência ou retorno que um fluxo disponibiliza, realiza, compartilha ou reinveste.

## 3. Categorias

| Código | Categoria | Exemplos |
|---|---|---|
| VO-01 | Orientação e clareza | interpretação, recomendação identificada, próximo passo |
| VO-02 | Oportunidade | vaga, benefício, parceria, experiência, conexão |
| VO-03 | Produto | bem físico ou digital |
| VO-04 | Serviço | atendimento, suporte, solução, logística |
| VO-05 | Experiência | viagem, evento, atividade, interação |
| VO-06 | Conteúdo e conhecimento | informação, formação, história, pesquisa |
| VO-07 | Conexão e acesso | introdução, comunidade, canal, disponibilidade |
| VO-08 | Conveniência e economia | redução de esforço, tempo, custo ou risco |
| VO-09 | Inteligência e insight | interpretação, personalização, previsão autorizada |
| VO-10 | Confiança e evidência | verificação, avaliação, histórico, rastreabilidade |
| VO-11 | Reputação e distribuição | visibilidade, audiência, alcance, reconhecimento contextual |
| VO-12 | Infraestrutura e capacidade | tecnologia, operação, financiamento, especialização |
| VO-13 | Benefício econômico | receita, remuneração, desconto, crédito contratual |
| VO-14 | Impacto social | recursos, mobilização, fortalecimento de causa |
| VO-15 | Aprendizagem | feedback, conhecimento produzido, melhoria do ecossistema |

## 4. Propriedades obrigatórias

Todo objeto deverá declarar:

- destinatário;
- fonte;
- proposta de benefício;
- condições de acesso;
- possibilidade de realização;
- evidência adequada;
- custos e riscos;
- dados envolvidos;
- autoridade e limites;
- validade ou duração, quando aplicável.

## 5. Objetos sensíveis

Exigem proteção reforçada:

- contexto pessoal;
- orientação sobre saúde, finanças, espiritualidade ou vulnerabilidade;
- acesso a comunidades sensíveis;
- reputação individual;
- evidências humanas;
- benefícios condicionados a comportamento;
- conteúdo patrocinado apresentado em contexto de decisão.

## 6. Separações

- dado autorizado pode contribuir para gerar valor, mas não é mercadoria livre;
- atenção pode ser objeto de distribuição, mas não equivale a consentimento;
- reputação pode ser retorno legítimo, mas não pode ser comprada como evidência;
- desconto é benefício econômico, não comprovação de qualidade;
- ponto é registro de incentivo contratual, não mérito humano;
- insight é apoio à decisão, não autoridade sobre o participante.

## 7. Estados aplicáveis

Um objeto poderá ser:

- candidato;
- elegível;
- disponibilizado;
- utilizado;
- expirado;
- contestado;
- corrigido;
- revogado;
- indisponível;
- encerrado.

Os estados específicos serão detalhados nos domínios de produto ou incentivo correspondentes.
