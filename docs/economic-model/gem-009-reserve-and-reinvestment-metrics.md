# Métricas de Reservas e Reinvestimento

## 1. Estados preservados

- reserva requerida conceitualmente;
- target proposto;
- target aprovado;
- recurso segregado;
- reserva constituída;
- reserva disponível;
- reserva comprometida;
- reserva utilizada;
- recomposição pendente.

Somente autoridade financeira e contábil futura poderá confirmar os estados monetários.

## 2. Indicadores candidatos

| Indicador | Fórmula conceitual |
|---|---|
| cobertura de obrigação | recursos elegíveis / obrigações protegidas elegíveis |
| cobertura da reserva | reserva disponível validada / target aprovado |
| velocidade de recomposição | valor recomposto / janela de recomposição |
| consumo de reserva | utilização elegível / reserva disponível inicial |
| aderência de reinvestimento | alocações compatíveis / alocações avaliadas |
| realização do reinvestimento | resultados observados / resultados esperados elegíveis |
| reversibilidade | alocações revisáveis / alocações avaliadas |

## 3. Regras

- reserva proposta não será exibida como constituída;
- recursos vinculados não cobrem finalidade incompatível;
- reinvestimento será rastreado da proposta ao resultado;
- distribuição ou expansão não precederá obrigações protegidas;
- retorno econômico não substituirá impacto, segurança ou continuidade.

## 4. Limite

Nenhuma fórmula define target, valor de reserva, política de investimento, alocação ou autorização de gasto.
