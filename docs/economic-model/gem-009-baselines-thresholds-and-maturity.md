# Baselines, Thresholds e Maturidade

## 1. Sequência obrigatória

```text
definição → fonte → instrumentação → observação → qualidade → baseline segmentada → hipótese de threshold → validação → aprovação → revisão
```

## 2. Tipos de referência

- baseline histórica interna;
- baseline por coorte ou segmento;
- comparação externa compatível;
- limite técnico;
- limite de risco;
- guardrail de tolerância zero;
- meta de melhoria;
- faixa operacional saudável.

Benchmark externo não substitui contexto ou baseline interna.

## 3. Maturidade

| Nível | Estado |
|---|---|
| M0 | pergunta ainda não definida |
| M1 | definição e contrato documentados |
| M2 | fonte e cálculo candidatos |
| M3 | observação inicial com qualidade conhecida |
| M4 | baseline segmentada e reproduzível |
| M5 | threshold validado e governado |
| M6 | uso decisório auditável e revisão contínua |

## 4. Regras

- esta versão mantém todas as métricas no máximo em M1;
- metas não serão derivadas de desejo estratégico isolado;
- thresholds incluirão janela, tolerância, exceção, autoridade e resposta;
- mudança de contexto poderá invalidar baseline;
- guardrail não será compensado por indicador agregado favorável;
- baixa maturidade deverá aparecer na mesma superfície que o resultado.
