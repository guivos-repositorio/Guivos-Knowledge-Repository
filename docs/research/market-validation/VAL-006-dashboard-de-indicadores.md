# VAL-006 — Dashboard de Indicadores de Validação

## 1. Objetivo

Definir o painel executivo para acompanhar a aceitação da proposta B2C da Guivos.

## 2. Indicadores principais

| Indicador | Cálculo | Meta inicial |
|---|---|---:|
| Dor percebida | % com resposta “frequentemente” ou “muito frequentemente”, complementada por casos moderados | >= 65% com dor moderada ou alta |
| Esforço atual | média da escala de esforço | >= 6,0 indica problema relevante |
| Compreensão autodeclarada | média da escala 0–10 | >= 8,0 |
| Compreensão correta | % de respostas abertas classificadas como corretas | >= 80% |
| Relevância percebida | média da escala 0–10 | >= 8,0 |
| ITP | média da pergunta de impacto em 12 meses | >= 8,0 |
| Confiança | média da escala 0–10 | >= 7,0 |
| Intenção de uso | % “provavelmente” + “certamente usaria” | >= 60% |
| Recorrência espontânea | % com motivo concreto de retorno | >= 50% |
| Interesse no beta | % “sim” | >= 35% |
| NPS conceitual | % promotores menos % detratores | >= 40 |
| Disposição para pagar | % que considera pagar, incluindo “depende do resultado” | diagnóstico, não gate isolado |

## 3. NPS conceitual

- promotores: notas 9 e 10;
- neutros: 7 e 8;
- detratores: 0 a 6;
- NPS = % promotores − % detratores.

O NPS conceitual mede recomendação da proposta, não satisfação com um produto em uso.

## 4. Painéis mínimos

### 4.1 Visão geral

- total de respostas válidas;
- status da amostra;
- índice geral de validação;
- semáforo por hipótese;
- decisão preliminar.

### 4.2 Problema

- frequência de perda de oportunidades;
- esforço para encontrar opções relevantes;
- canais atuais;
- principais dores abertas.

### 4.3 Proposta

- compreensão;
- relevância;
- ITP;
- elemento de maior valor;
- partes confusas.

### 4.4 Confiança

- nota média;
- fatores que aumentam confiança;
- dados que as pessoas não desejam compartilhar;
- principais barreiras.

### 4.5 Adoção

- intenção de experimentar;
- frequência esperada;
- primeiro uso imaginado;
- recorrência espontânea;
- interesse no beta.

### 4.6 Monetização

- disposição para pagar;
- faixas de preço;
- condições para considerar o pagamento justo.

## 5. Índice Geral de Validação

O índice geral poderá ser calculado como referência, sem substituir a leitura por hipótese:

- dor: 20%;
- compreensão: 20%;
- relevância e ITP: 25%;
- confiança: 15%;
- intenção de uso e recorrência: 15%;
- beta: 5%.

Monetização será exibida separadamente na primeira rodada.

## 6. Segmentações

Todos os principais indicadores deverão permitir corte por idade, situação profissional, região, intensidade da dor e interesse principal.
