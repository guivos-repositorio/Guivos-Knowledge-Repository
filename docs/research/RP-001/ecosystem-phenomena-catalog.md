# RP-001 — Ecosystem Phenomena Catalog

## Objetivo

Consolidar fenômenos e condições recorrentes identificados durante o RP-001 antes de sua tradução em Candidate Outcomes.

## Regra de uso

O catálogo registra fenômenos observados e sintetizados. Nenhum item deste documento é automaticamente um Ecosystem Outcome ou integrante da Canon.

## Estrutura dos registros

| Campo | Descrição |
|---|---|
| ID | Identificador provisório |
| Fenômeno | Nome do fenômeno |
| Definição de trabalho | Síntese provisória |
| Disciplinas de origem | Áreas em que aparece |
| Evidências relacionadas | IDs do Evidence Registry |
| Recorrência | Frequência observada |
| Relevância para a Guivos | A avaliar |
| Status | Candidate, Accepted, Merged, Rejected ou Deferred |

## Fenômenos

Nenhum fenômeno consolidado até o momento.
