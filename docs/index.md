# Guivos Knowledge Repository

O GKR é a fonte oficial de conhecimento da Guivos.

Ele existe para preservar, organizar e versionar o patrimônio intelectual da empresa de forma independente de conversas, ferramentas ou documentos isolados.

## Conteúdo

- Guivos Ecosystem Blueprint (GEB)
- Architecture Decision Records (ADRs)
- Glossário oficial
- Diagramas
- Roadmap documental
