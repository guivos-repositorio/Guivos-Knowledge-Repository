# Guivos Knowledge Repository

O GKR é a fonte oficial, versionada e governada do conhecimento da Guivos.

Ele preserva fundamentos, arquiteturas, modelos, decisões, pesquisas, validações e publicações públicas.

## Estado atual

- Foundation congelada em `A2-B3`;
- A2-R02 — Fundamental Model Review ativa;
- `A2-R02-FMEM-001` pronta para execução;
- Guivos Mall consolidado como produto oficial;
- Guivos Economic Model criado como domínio planejado;
- Guia Oficial da Guivos na versão 3.1.0.
