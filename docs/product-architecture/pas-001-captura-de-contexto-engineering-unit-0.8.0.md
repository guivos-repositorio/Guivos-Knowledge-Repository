## 3. Modelo mínimo

A estrutura mínima permite:

- uma pessoa para Architecture + Engineering + Platform/DevOps provisório;
- uma pessoa independente para Security & Privacy;
- uma pessoa independente para Quality & Evidence;
- Data Owner antes do W0-03.

Nenhuma pessoa poderá acumular todos os papéis. Security e Quality permanecem independentes de Engineering.

- EV-017 de owners;
- autorização técnica.

## 5. Próxima decisão permitida
