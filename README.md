## Princípio de autoridade

O GKR é a fonte oficial. Conversas, protótipos, arquivos legados e materiais externos são evidências; não substituem documentos canônicos. Implementações técnicas deverão preservar as decisões funcionais e retornar ao contrato competente diante de conflito.
